Art In Color


When I go to museums, I find a lot of inspiration from the colors in artworks. Art In Color uses the Google Cloud Vision API to generate color palettes for a number of artworks in the public domain. People can copy hex codes to use in their own projects with a single click, or feed an image URL to generate their own color palettes.
